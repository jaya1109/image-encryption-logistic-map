🔐 Image Encryption & Decryption

Using Logistic Map | Cyber Security Project

📌 Project Overview

This project secures medical images using a logistic map-based chaotic encryption technique. Images are converted into an unreadable encrypted format and restored using a secret key.

🎯 Objectives

  • Secure medical image transmission
  • Apply chaos theory in encryption
  • Prevent unauthorized access

🧠 Concept

The logistic map generates pseudo-random chaotic sequences used for encryption. Formula: x(n+1) = r * x(n) * (1 - x(n))

⚙️ Working Process

  • Input Image
  • Grayscale Conversion
  • Logistic Sequence Generation
  • Encryption (XOR / Pixel Shuffle)
  • Decryption using same key
